# 🧠 Memory Card Game

A simple and fun **Memory Card Game** built using modern web technologies. Flip cards, match pairs, and test your memory skills!

---


## 🌐 Live Demo

👉 https://jaykumar122.github.io/Memory-Card-Game/

---
## 🎮 Features

* 🃏 Flip cards to reveal hidden symbols
* 🔍 Match pairs of identical cards
* ⏱️ Track your moves or time (optional feature)
* 🎨 Clean and responsive UI
* 🔄 Restart / reset game anytime

---

## 🚀 How to Play

1. Click on any card to flip it
2. Flip another card to find a matching pair
3. If both cards match → they stay open ✅
4. If not → they flip back ❌
5. Continue until all pairs are matched

## 📂 Project Setup

Clone the repository:

```bash
git clone https://github.com/Jaykumar122/Memory-Card-Game.git
```

Go to the project folder:

```bash
cd memory-card-game
```

Run the project:

* Open `index.html` in your browser
  **or**

```bash
npm install
npm run dev
```
